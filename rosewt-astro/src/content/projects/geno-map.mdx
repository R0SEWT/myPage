## Sistema

Framework de validación para datos de alta dimensionalidad cuando no existe correspondencia explícita entre muestras. Evalúa estabilidad estructural de representaciones mediante invariantes de grafos kNN en lugar de depender de alineaciones punto a punto.

## Enfoque

**Construcción de grafos kNN** → **perturbación controlada** → **medición de invariantes estructurales** → **comparación entre métodos de representación**.

## Hallazgos

- La estructura de vecindad se degrada de forma continua incluso bajo perturbaciones severas
- PCA preserva mejor la estabilidad estructural que autoencoders en este régimen
- UMAP cambia la visualización, pero no el grafo analítico subyacente

**Stack:** PCA, UMAP, grafos kNN, Python
