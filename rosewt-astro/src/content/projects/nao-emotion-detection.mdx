## Sistema

Sistema de reconocimiento de emociones faciales implementado en robot NAO, diseñado para intervención psicoeducativa con niños con TEA (Trastorno del Espectro Autista). Combina computer vision, deep learning y robótica social.

## Arquitectura

Pipeline end-to-end: **captura** (cámara NAO) → **transmisión** base64 HTTP → **Flask API** → **inferencia CNN** → **respuesta adaptativa** del robot (voz + gestos). Latencia total <120ms.

## Decisiones técnicas

- **Comparación de 3 arquitecturas** CNN (MobileNetV2, VGG19, custom) bajo restricciones de hardware — MobileNetV2 seleccionado por tradeoff accuracy/latencia (89% accuracy, 15ms inferencia)
- **150+ trials Optuna** con búsqueda multi-objetivo (accuracy vs. tiempo de inferencia) y median pruning
- **HTTP RESTful** sobre ROS/sockets: simplicidad de deployment y compatibilidad con Python 2.7 del NAO
- **Transfer learning** con ImageNet + cross-validation 5-fold + data augmentation para mitigar dataset limitado

## Validación

- 15 participantes en sesiones controladas en FabLab (45 min cada una)
- 91% detección correcta bajo condiciones reales de iluminación
- 7 emociones soportadas: happy, sad, angry, surprise, fear, disgust, neutral

## Restricciones

- Hardware limitado del NAO (Python 2.7, pynaoqi SDK)
- Dataset FER2013 con ruido y variabilidad inherente en expresiones humanas
- Requisito de latencia <120ms para interacción natural con niños

**Stack:** PyTorch, Flask, OpenCV, MediaPipe, Optuna, Python
