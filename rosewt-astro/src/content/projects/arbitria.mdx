Sistema RAG (Retrieval-Augmented Generation) especializado en documentos legales de arbitraje. 

**Características principales:**
- 🔍 **Chunking inteligente**: Procesamiento optimizado para documentos PDF complejos
- 🧠 **Embeddings semánticos**: Búsqueda contextual avanzada con OpenAI
- 💬 **Chat contextual**: Interfaz conversacional para consultas jurídicas
- ⚡ **API REST**: Backend escalable con FastAPI
- 📊 **Pipeline optimizado**: Procesamiento eficiente de grandes volúmenes de documentos

**Stack técnico:** Python, LlamaIndex, FastAPI, OpenAI API, PostgreSQL, Docker
