## Sistema

Sistema de predicción de riesgo de colelitiasis orientado a screening en contextos con restricciones de infraestructura. Replantea el problema para operar sin depender de variables clínicas que no siempre están disponibles fuera de entornos hospitalarios.

## Enfoque

**Selección de variables viables en campo** → **entrenamiento y tuning** → **comparación entre escenarios observables** → **inspección individual con SHAP**.

## Resultado

- Degradación controlada al reducir el espacio de variables disponibles
- Evaluación explícita entre performance y viabilidad operativa
- Interfaz para revisar predicciones y sensibilidad de features caso por caso

**Stack:** XGBoost, SHAP, Optuna, Python
