Sistema de variantes

ARV · PATT · 01

El toggle Andino/FMA es una decisión central del proyecto: el mismo sistema bajo dos tesis estéticas, comparables en vivo. Todo cambio de paleta pasa por variables CSS, nunca por duplicación de código.

Variante · default

70% Andino

Carbón + hueso + cochinilla. El sistema por defecto. Tono archival andino con alquimia como acento.

data-variant="andino"

Variante · alterna

70% FMA

Pergamino + tinta oro + rojo sangre. Desplazamiento hacia alquimia occidental — para folios rituales o de investigación filosófica.

data-variant="fma"

El toggle 70% Andino / 70% FMA vive fijo en la esquina superior derecha. Toda la paleta cambia vía CSS vars en 0.8s — sin flash, sin recarga. El usuario puede comparar la misma página bajo dos tesis visuales.
