Arariwa es esquinas vivas. Sin redondeo. El único radio del sistema es 2px y sólo aparece en controles pequeños. Los sellos son circulares porque son sellos, no UI.

--radius-0 · 0
Default. Especímenes, fichas, tarjetas de proyecto.

Toggles, tags, controles pequeños. Único radio permitido.

circle · seal only
Reservado para sigilos y sellos. No usar en UI.

Regla: esquinas vivas por defecto. Cualquier redondeo mayor a 2px rompe la voz archival — evita "tarjeta SaaS con acento izquierdo".
